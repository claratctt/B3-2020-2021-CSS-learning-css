Propriétés du CSS

Inline Block

Un élément mis en "Inline block" est très similaire à l'inline en ce sens qu'il se met en ligne avec le flux naturel du texte (sur la "ligne de base"). La différence est que vous pouvez définir une largeur et une hauteur qui seront respectées.

CSS Columns

Avec les colonnes CSS, vous pouvez créer une mise en page inspirée de l'impression avec peu de balisage supplémentaire qui peut s'adapter au-delà d'un canevas fixe. Un navigateur pris en charge effectuera des calculs pour envelopper et équilibrer le contenu dans des colonnes bien ordonnées. Si vous travaillez déjà avec une mise en page fluide, les colonnes se recomposeront automatiquement. Avec la bonne combinaison de propriétés, les colonnes CSS peuvent constituer une option de mise en page intéressante, qui est conviviale tout en se dégradant avec grâce.

Flexbox

Avec les flexbox, on définit un conteneur à l'intérieur duquel on va placer plusieurs éléments. Et avec flexbox on peut agencer les éléments comme on le souhaite.

Media queries

Les requêtes média (media queries) permettent de modifier l'apparence d'un site ou d'une application en fonction du type d'appareil (impression ou écran par exemple) et de ses caractéristiques (la résolution d'écran ou la largeur de la zone d'affichage (viewport) par exemple).

CSS grid layout

Le module CSS Grid layout (modèle de disposition en grille) est un module de la spécification CSS qui permet de créer des mises en page en divisant l'espace d'affichage en régions utilisables par une application ou en définissant des relations de taille? position et d'empilement entre les éléments HTML.

Object fit

La propriété object-fit définit la façon dont le contenu d'une image ou vidéo doit s'adapteretse redimensionner à son conteneur en utilisant sa largeur et sa hauteur.
