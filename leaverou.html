Léa Verou

Lea Verou est une geek des standards du web / front-end développeur / web designer / conférencière / auteure originaire de Grèce. Lea a étudié l'informatique à l'université d'économie et de commerce d'Athènes, où elle a coorganisé et parfois donné un cours de développement Web pour les étudiants de 4e année. Dans le passé, elle a travaillé pour le W3C/MIT. Elle a été la co-fondatrice d'une startup grecque appelée Fresse Ltd , qu'elle a quitté en 2011. Aussi, elle a écrit un livre sur les CSS avancés pour O'Reilly. Elle est passionnée par les standards du web, en particulier JavaScript et CSS. Elle crée des outils qui aident les autres développeurs à travailler avec ces standards, et en parle dans des conférences partout dans le monde. Lea écrit sur les standards dans les médias populaires de l'industrie et sur les blogs concernant ses recherches et ses outils. Elle participe également au développement du CSS en tant qu'experte invitée du groupe de travail CSS.
