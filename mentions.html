Techniques utilisées

J'ai créee ce site dans le cadre d'un projet d'études à l'ECV Digital de Bordeaux afin de apprendre les bases du CSS.

Ce site est donc conçu en HTML et CSS avec une structure "Barebones V3" , un "squelette" basé sur le projet Skeleton de Dave Gamache.

J'ai été amenée à utiliser différentes techniques CSS pour construire ce site: Css grid pour la mise en page des personnalités; Flebox pour la fiche biographique de Lea Verou; et la propriété object-fit ainsi que des miedia queries pour la mise en page des cartes des différentes personnalités suivant la taille du device utilisé.

J'ai aussi utilisé une variable font pour la typographie des titres.
